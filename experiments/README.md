# Experiments

这里放可复现的实验记录。

每个实验建议单独一个目录：

```text
experiments/
└── 2026-07-agent-eval-demo/
    ├── README.md
    ├── commands.md
    └── results.md
```

实验必须写清楚：

- 环境和依赖。
- 输入数据。
- 执行命令。
- 结果。
- 失败点。
- 下一步改进。
